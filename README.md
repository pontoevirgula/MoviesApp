# MOVIES APP
Projeto que acessa IMDB API

-  Dagger Hilt como injeção de dependência
-  Arquitetura MVVM + Clean Arch
-  Multi modular
-  Retrofit2
-  Coroutines
-  Room DB
-  Jetpack Compose
-  Navigation Component
-  Coil pra manipulação de imagens
